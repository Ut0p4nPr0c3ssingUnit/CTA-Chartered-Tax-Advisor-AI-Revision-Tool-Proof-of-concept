## 📋 Overview

This tool uses Google's Vertex AI and Gemini 1.5 Pro to:
- Parse Tolley's tax reference materials
- Generate interactive flashcards with:
  - Clear revision questions
  - Precise statutory answers
  - Visual layout hints for better retention
  - Mnemonic anchors for memory aid
- Render a beautiful interactive HTML frontend with flip animations

## 🔑 Key Features

- **Schema-Enforced Generation**: Structured JSON output for consistent flashcard format
- **Tax-Specific**: Focused on UK Chartered Tax Advisor (CTA) examination topics
- **Dynamic Prompting**: Customize flashcard generation by topic
- **Interactive Visualizer**: HTML5 frontend with flip animations and session tracking
- **GCP Integration**: Automatic project detection and Cloud Storage integration

## 📝 Prerequisites

- Google Colab environment (recommended)
- Google Cloud Project with Vertex AI API enabled
- Active GCP billing account
- Tolley's reference PDF file

## 🎯 How to Use

1. Click the "Open in Colab" button above
2. Run the cells in sequence
3. Authenticate with your Google Cloud account
4. Upload or specify your Tolley's reference PDF
5. Enter the tax topic you want to focus on
6. Generate interactive flashcards
7. View and track your revision progress in the interactive visualizer
